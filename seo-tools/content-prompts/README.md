# SEO Content Generation Prompts

A collection of proven prompt templates for generating SEO-optimized content for EuroCheck and similar products.

## Contents

1. **Blog Post Templates** — Various formats for different intents
2. **Landing Page Copy** — Conversion-focused page content
3. **Product Descriptions** — Feature highlights and benefits
4. **FAQ Generation** — Common questions and schema-ready answers
5. **Meta Tags** — Titles, descriptions, and social media

## Usage

Each prompt is designed to be copy-pasted into Claude, GPT-4, or similar LLMs. Replace `[PLACEHOLDERS]` with your specific information.

## Files

```
content-prompts/
├── README.md           # This file
├── blog-posts.md       # Blog post templates
├── landing-pages.md    # Landing page copy prompts
├── meta-tags.md        # SEO meta content prompts
└── faq-schema.md       # FAQ generation with schema
```

## Tips

- Always include target keyword naturally 2-3 times
- Focus on search intent (informational, transactional, navigational)
- Use proper heading hierarchy (H1 → H2 → H3)
- Include internal links to related content
- Add relevant external authority links
