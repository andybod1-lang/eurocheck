# Blog Post Prompt Templates

## 1. How-To Guide (Informational Intent)

```
Write a comprehensive how-to guide for [TOPIC].

Target keyword: [PRIMARY_KEYWORD]
Secondary keywords: [KEYWORD_2], [KEYWORD_3], [KEYWORD_4]

Requirements:
- Title: Engaging H1 with keyword near the start
- Introduction: Hook + problem statement + what reader will learn (150 words)
- Main content: 5-7 actionable steps with subheadings (H2)
- Each step: 100-150 words with practical advice
- Include: Pro tips, common mistakes to avoid, real examples
- Conclusion: Summary + CTA to [PRODUCT/SERVICE]
- Word count: 1,500-2,000 words

Tone: Helpful, authoritative, conversational
Audience: [TARGET_AUDIENCE]

Format output with proper markdown headings.
```

## 2. Listicle / Roundup (Comparison Intent)

```
Create a listicle article: "Top [NUMBER] [CATEGORY] in [YEAR]"

Target keyword: [PRIMARY_KEYWORD]
Secondary keywords: [KEYWORD_2], [KEYWORD_3]

Structure:
1. Introduction: Why this matters + selection criteria (100 words)
2. Quick comparison table with key features
3. Each item (200-300 words):
   - Name and brief description
   - Key features (bullet points)
   - Pros and cons
   - Best for (use case)
   - Pricing if applicable
4. How we selected these + methodology
5. FAQ section (3-5 questions)
6. Conclusion with recommendation

Include [OUR_PRODUCT] as one option with honest positioning.
Word count: 2,500-3,500 words
```

## 3. Problem-Solution Post (Pain Point Focus)

```
Write a blog post addressing: "[PAIN_POINT_QUESTION]"

Target keyword: [PRIMARY_KEYWORD]
Related keywords: [KEYWORD_2], [KEYWORD_3]

Structure:
1. Empathize with the problem (why it's frustrating)
2. Explain why common solutions don't work
3. Introduce the better approach
4. Step-by-step solution
5. How [PRODUCT] helps solve this
6. Results readers can expect
7. Quick start guide

Tone: Understanding, solution-focused, practical
Length: 1,200-1,800 words

Start with a relatable scenario that hooks the reader.
```

## 4. Ultimate Guide (Pillar Content)

```
Create an ultimate guide to [TOPIC].

Target keyword: [PRIMARY_KEYWORD]
Cluster keywords: [KEYWORD_2], [KEYWORD_3], [KEYWORD_4], [KEYWORD_5]

This should be comprehensive pillar content covering:

1. Introduction
   - What is [TOPIC]
   - Why it matters
   - What you'll learn

2. Fundamentals (Chapter 1)
   - Core concepts
   - Key terminology
   - Common misconceptions

3. Getting Started (Chapter 2)
   - Prerequisites
   - First steps
   - Beginner mistakes to avoid

4. Intermediate Strategies (Chapter 3)
   - Advanced techniques
   - Tools and resources
   - Case studies

5. Expert Level (Chapter 4)
   - Pro tips
   - Optimization strategies
   - Future trends

6. Resources
   - Recommended tools
   - Further reading
   - Community links

Word count: 4,000-6,000 words
Include internal links to related articles: [LINK_1], [LINK_2]
Add a downloadable checklist or template offer.
```

## 5. Case Study Post

```
Write a case study blog post about [RESULT/ACHIEVEMENT].

Subject: [COMPANY/PERSON] achieved [SPECIFIC_RESULT] using [PRODUCT/METHOD]

Structure:
1. Headline: Specific result with numbers
2. Executive summary: Key takeaway in 2-3 sentences
3. The Challenge
   - Background
   - Problem they faced
   - Previous attempts
4. The Solution
   - Why they chose [APPROACH]
   - Implementation process
   - Timeline
5. The Results
   - Specific metrics (before/after)
   - Unexpected benefits
   - Testimonial quote
6. Key Lessons
   - What worked
   - What they'd do differently
   - Advice for others
7. CTA: How readers can achieve similar results

Include data visualizations where possible.
Length: 1,500-2,000 words
```

## 6. News/Trend Analysis

```
Write an analysis piece about [RECENT_DEVELOPMENT/TREND].

Target keyword: [PRIMARY_KEYWORD]

Include:
1. What happened (factual summary)
2. Why it matters
3. Industry reaction
4. Impact on [TARGET_AUDIENCE]
5. What to do about it (actionable advice)
6. Our take / expert opinion
7. What to watch next

Tone: Timely, authoritative, insightful
Length: 800-1,200 words
Cite sources where appropriate.
```

## 7. Beginner's Guide

```
Write a beginner-friendly introduction to [TOPIC].

Target keyword: [PRIMARY_KEYWORD] + "for beginners"
Secondary: [KEYWORD_2], [KEYWORD_3]

Audience: Complete beginners with no prior knowledge

Structure:
1. What is [TOPIC]? (Simple explanation)
2. Why should you care?
3. Key concepts explained simply
4. Step-by-step getting started guide
5. Common questions answered
6. Next steps / resources
7. Glossary of terms

Use:
- Simple language (no jargon without explanation)
- Analogies and examples
- Visual descriptions
- Encouraging tone

Length: 1,800-2,500 words
```

## Usage Tips

1. **Customize placeholders** before using
2. **Add context** about your brand voice
3. **Include examples** of successful content
4. **Specify constraints** (avoid certain words, required phrases)
5. **Request alternatives** if first output isn't right
