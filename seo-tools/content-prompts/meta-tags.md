# Meta Tags & SEO Copy Prompts

## 1. Title Tags

```
Generate 5 SEO title tag variations for this page:

Page topic: [TOPIC]
Primary keyword: [KEYWORD]
Brand name: [BRAND]
Page type: [homepage/product/blog/landing]

Requirements:
- 50-60 characters including spaces
- Keyword near the beginning
- Brand at end (if fits)
- Compelling for clicks

Include:
- One question format
- One number/list format
- One benefit-focused
- One action-oriented
- One unique angle
```

## 2. Meta Descriptions

```
Write 5 meta description variations for:

Page: [PAGE_TITLE]
Primary keyword: [KEYWORD]
Target audience: [AUDIENCE]
Main CTA: [WHAT_SHOULD_THEY_DO]

Requirements:
- 150-160 characters
- Include keyword naturally
- Clear value proposition
- End with CTA or hook
- Unique selling point

Variations:
1. Benefit-focused
2. Problem-solution
3. Social proof angle
4. Urgency/scarcity
5. Question hook
```

## 3. Open Graph / Social Media

```
Create Open Graph tags for:

Page URL: [URL]
Page type: [article/website/product]
Primary keyword: [KEYWORD]

Generate:
1. og:title (different from page title, optimized for social sharing)
2. og:description (compelling for social, different from meta)
3. Twitter card title
4. Twitter card description
5. Image alt text suggestion

Social media optimization:
- More conversational tone
- Emoji usage if appropriate
- Hashtag suggestions
```

## 4. Heading Structure

```
Create an SEO-optimized heading structure for:

Topic: [TOPIC]
Primary keyword: [KEYWORD]
Secondary keywords: [KEYWORD_2], [KEYWORD_3], [KEYWORD_4]
Content length: [WORD_COUNT]

Generate:
- H1: One main heading (keyword near start)
- H2s: 4-6 section headings (include secondary keywords)
- H3s: 2-3 under each H2 where appropriate

Requirements:
- Natural language (not keyword stuffed)
- Logical flow
- Scannable structure
- Mix of statement and question formats
```

## 5. URL Slug Optimization

```
Suggest optimized URL slugs for these pages:

Pages:
1. [PAGE_DESCRIPTION_1]
2. [PAGE_DESCRIPTION_2]
3. [PAGE_DESCRIPTION_3]

Requirements:
- Short (3-5 words ideal)
- Include primary keyword
- No stop words unless necessary
- Lowercase, hyphen-separated
- Evergreen (no dates unless time-sensitive)

For each, provide:
- Recommended slug
- Alternative option
- Why this structure
```

## 6. Image Alt Text

```
Write SEO-friendly alt text for these images:

Page topic: [TOPIC]
Primary keyword: [KEYWORD]

Images:
1. [IMAGE_DESCRIPTION_1]
2. [IMAGE_DESCRIPTION_2]
3. [IMAGE_DESCRIPTION_3]

Requirements:
- Descriptive but concise (125 characters max)
- Include keyword naturally (not forced)
- Describe what's IN the image
- Useful for screen readers

Format: "Alt: [description]"
```

## 7. Schema Markup Content

```
Generate content for schema markup:

Page type: [PRODUCT/ARTICLE/FAQ/LOCAL_BUSINESS/SOFTWARE]
Topic: [TOPIC]

For Product schema:
- Name
- Description (150-300 chars)
- Brand
- SKU/identifier
- Review snippet text

For Article schema:
- Headline
- Description
- Key points (for speakable)

For FAQ schema:
- 5 questions with answers
- Conversational Q&A format
- Include target keywords naturally

For Software schema:
- Application name
- Description
- Category
- Operating system
- Offer details
```

## 8. Internal Link Anchor Text

```
Suggest anchor text variations for internal links to:

Target page: [PAGE_TITLE]
Target keyword: [KEYWORD]
Page URL: [URL]

Context: These links will appear in:
1. Blog posts
2. Related content sections
3. Navigation
4. In-content mentions

Generate 10 anchor text variations:
- 3 exact match keyword
- 3 partial match
- 2 branded
- 2 natural/contextual

Avoid:
- "Click here"
- "Read more"
- Over-optimization (same anchor everywhere)
```

## 9. Featured Snippet Optimization

```
Optimize this content for featured snippets:

Target query: "[QUESTION_KEYWORD]"
Current answer: [EXISTING_CONTENT]

Generate:
1. Definition box (40-60 words, starts with "[TERM] is...")
2. Step list (numbered, 5-8 steps, each <15 words)
3. Bullet list (5-8 items)
4. Table format (if applicable)
5. Comparison format (X vs Y)

Requirements:
- Direct answer to query
- Formatted for snippet extraction
- Includes target phrase
- Concise but complete
```

## 10. Local SEO Content

```
Generate local SEO content for:

Business: [BUSINESS_NAME]
Service: [SERVICE]
Location: [CITY, STATE]
Service areas: [NEARBY_AREAS]

Generate:
1. Location page title tag
2. Location page meta description
3. Location page H1
4. NAP (Name, Address, Phone) format
5. Local business schema description
6. Service area content (mentions nearby cities)
7. Local keywords to target (5-10)
8. Google Business Profile description (750 chars)
```

## Quick Reference: Character Limits

| Element | Optimal | Maximum |
|---------|---------|---------|
| Title tag | 50-60 | 70 |
| Meta description | 150-160 | 320 |
| URL slug | 3-5 words | 75 chars |
| H1 | 20-70 chars | 100 |
| Alt text | 80-125 | 125 |
| OG title | 40-60 | 90 |
| OG description | 100-150 | 200 |
| Tweet | 71-100 | 280 |
