## Privacy Commitment

EuroCheck was built with privacy as a core principle, not an afterthought:

- **No tracking whatsoever** - We don't know who uses it or how
- **No external requests** - The extension never phones home
- **No account required** - Install and use, nothing else
- **Minimal permissions** - Only activates on e-commerce sites
- **Open source** - Every line of code is auditable
