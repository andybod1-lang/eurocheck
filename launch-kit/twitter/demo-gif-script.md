## Recording Tips

**Browser setup:**
- Use a clean Chrome/Firefox profile
- Hide bookmarks bar or use presentable bookmarks
- Set zoom to 100%
- Use incognito/private mode for clean state

**Recording tools:**
- macOS: Built-in screen recording (Cmd+Shift+5)
- GIF conversion: gifski, ffmpeg, or online tools
- Editing: Kapwing, ezgif.com

**Quality tips:**
- Record at 2x resolution, scale down
- Avoid fast mouse movements (looks janky in GIF)
- Pause slightly at key moments
- Keep cursor movements smooth and intentional
